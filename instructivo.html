Descripción General del Sistema

¿Qué hace este sistema?

Parqueadero La Esperanza es una aplicación web completa para la gestión de acceso vehicular en conjuntos residenciales. Permite controlar entradas y salidas con verificación automática de restricciones, OCR simulado para lectura de placas, y generación de tiquetes.

React SPA Sin backend requerido 100% navegador Datos en memoria

Instalación y Puesta en Marcha

Requisitos previos
  • Node.js versión 18 o superior — nodejs.org
  • npm 9+ (incluido con Node.js)
  • Navegador moderno: Chrome, Edge, Firefox o Safari
  • Conexión a internet (solo para fuentes de Google en desarrollo)
Pasos de instalación
1

Crear el proyecto

Abre una terminal en la carpeta donde quieres guardar el proyecto y ejecuta:

npm create vite@latest parqueadero-la-esperanza -- --template react

cd parqueadero-la-esperanza
2

Instalar dependencias

npm install
3

Reemplazar archivos

Copia los archivos descargados (App.jsx, main.jsx, index.css, index.html) en las rutas correspondientes dentro de la carpeta del proyecto. Reemplaza los existentes.

4

Iniciar en desarrollo

npm run dev

Abre el navegador en http://localhost:5173

5

Compilar para producción

npm run build

Los archivos listos quedan en la carpeta dist/. Puedes subirlos a cualquier servidor web estático (Nginx, Apache, Vercel, Netlify).

ℹ️
Modo demo: El sistema funciona completamente sin base de datos. Los datos están en memoria (se reinician al recargar la página). Para persistencia real, integrar con PHP/MySQL o SQLite como backend.

Guía de Uso — Control de Acceso

Flujo de ingreso de un vehículo
1

Captura de placa

Haga clic en 📸 Capturar OCR para simular la lectura de placa por cámara. En producción, conecte la cámara IP al módulo OCR real (OpenALPR / EasyOCR).

2

Verificación automática

El sistema consulta si la placa está registrada y evalúa: Mora de mensualidades Restricción de convivencia Pico y placa activo

3

Seleccionar tipo de movimiento

Elija ENTRADA o SALIDA en el selector.

4

Registrar

Haga clic en ✅ Registrar ENTRADA/SALIDA. El sistema guarda el movimiento y emite una notificación.

5

Imprimir tiquete

Haga clic en 🖨️ Imprimir Tiquete para enviar a la impresora POS térmica (en producción usa la API de impresión del navegador).

Personalización del Sistema

Cambiar nombre del conjunto / parqueadero

En App.jsx, busca todas las ocurrencias de "PARQUEADERO LA ESPERANZA" y "Parqueadero La Esperanza" y reemplázalas con el nombre real.

Modificar calendario de Pico y Placa

Edita el objeto PICO al inicio de App.jsx. Las claves son el día de la semana (0=Dom, 1=Lun… 6=Sáb) y el valor es un array con los dígitos restringidos:

const PICO = { 1:[3,4], 2:[5,6], 3:[7,8], 4:[9,0], 5:[1,2], 0:[], 6:[] };
Agregar vehículos iniciales

Edita el array DB.vehiculos al inicio de App.jsx con los datos reales del conjunto residencial.

Número de celdas

Cambia el número 24 en Array.from({length:24},...) dentro de DB.celdas por el número real de celdas del parqueadero.

⚠️
Integración OCR real: Para conectar una cámara IP con reconocimiento de placas real, reemplaza la función simularOCR() con una llamada a la API de OpenALPR, EasyOCR o la solución OCR del cliente.
